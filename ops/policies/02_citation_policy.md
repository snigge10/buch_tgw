# Citation Policy (Fußnoten)

## Format (Markdown)
Im Text:
- Aussage ...[^fn-1]

Fußnote:
- [^fn-1]: Autor/Organisation, Titel, Datum (oder Jahr), URL/DOI, Abrufdatum (YYYY-MM-DD).

## Regeln
- Jede faktische Behauptung -> mindestens eine Fußnote.
- Zahlen/Statistiken -> Fußnote + idealerweise Primärquelle.
- Keine verbotenen Domains.
- Keine langen Direktzitate (max. 25 Wörter).
